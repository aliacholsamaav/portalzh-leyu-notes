# portalzh-leyu-notes

## 项目简介

这是一个用于归档和发布多个独立 HTML 页面的仓库。  
仓库中的页面内容以笔记、记录和参考文档为主，不针对任何特定网站或域名。  
适合需要快速浏览或引用离线化、轻量级页面的场景。

## 目录说明

```
portalzh-leyu-notes/
├── README.md           # 本文件
├── pages/              # HTML 页面存放目录（按主题或时间组织）
│   ├── example-1.html  # 示例页面 1
│   ├── example-2.html  # 示例页面 2
│   └── ...
└── assets/             # 页面引用的静态资源（如图片、样式、脚本）
```

- `pages/`：所有独立 HTML 页面的主目录，页面文件名应具有描述性。
- `assets/`：存放页面中引用的图片、CSS、JavaScript 等资源，便于统一管理。

## 页面归档说明

1. **页面独立性**：每个 HTML 文件均独立自包含，可脱离仓库环境直接打开使用。
2. **归档范围**：收录个人学习、工作或兴趣相关的笔记类页面，不包含任何商业或推广内容。
3. **内容来源**：页面内容由作者自行整理或创作，不转载他人作品。
4. **更新频率**：不定期新增或修订页面，无固定发布周期。
